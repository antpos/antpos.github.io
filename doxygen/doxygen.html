What is doxygen

Doxygen[1] is a tool for creating documentation from the source files. The documentation is written within the sources and doxygen uses these to create the documentation.
Doxygen is releleased under GNU General Public License version 2[2].

Getting started

Easiest way to install Doxygen[1] is trough repositpories.

apt-get update
apt-get install doxygen

Use sudo if necessary. Other possibility is to compile it directly from source. Source can easily pulled from GitHub[3] repository.
